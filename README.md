## Lab 1 - 1D Unconstrained Optimization

Main objective: compare interval-reduction methods for scalar minimization.

Algorithms implemented:
- Bisection method using function evaluations around midpoint (`bisection.m`)
- Bisection method using derivative sign (`bisection_derivatives.m`)
- Fibonacci search (`fibonacci.m`)
- Golden section search (`golden_section.m`)

Typical analysis included in code:
- Parameter sweeps over stopping interval length and method parameters
- Function-evaluation counting per method
- Convergence visualization through objective evaluations vs parameter choices
- Convergence visualization through interval endpoints $a_k$ and $b_k$ vs iteration $k$

Test functions are explicitly defined in each script and compared under common initial intervals.

## Lab 2 - Multivariate Unconstrained Optimization

Main objective: study first-order and second-order methods on a nonlinear 2D objective.

Algorithms implemented:
- Steepest descent (`steepest_descent.m`)
- Newton method (`newton.m`)
- Levenberg-Marquardt variant for non-positive-definite Hessian cases (`levenberg_marquardt.m`)

Step-size strategies explored:
- Constant step size
- Armijo rule
- Line-search by minimizing along direction with derivative-based bisection

Key outputs generated by scripts:
- Surface and contour visualizations of the objective
- Convergence plots of $f(x_k, y_k)$ over iterations
- Comparative behavior across different initial points

## Lab 3 - Constrained Optimization and Projection

Main objective: examine behavior of gradient methods with fixed steps and constraints.

Algorithms implemented:
- Steepest descent with constant step sizes on a quadratic objective (`steepest_descent.m`)
- Projected steepest descent with box constraints (`steepest_descent_projection.m`)

Constrained setup details included in code:
- Projection operator for coordinate-wise bounds
- Iterative update using projected intermediate point
- Convergence comparison for multiple initial points and parameter choices

Extra diagnostics:
- Linear-scale and logarithmic-scale convergence plots for stability/divergence interpretation

## How To Run

Prerequisites:
- MATLAB (or compatible environment such as GNU Octave for most scripts)

Basic usage:
1. Open a specific script from a lab's `Matlab` folder.
2. Run the script from the MATLAB editor.
3. Inspect generated figures for convergence and method comparison.

Notes:
- Scripts are self-contained and define test functions internally.
- Most scripts execute multiple experiments sequentially and open several figures.
